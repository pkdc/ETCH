Work one phase per session. Have Claude Code append cross-cutting findings to a running `NOTES.md` at the repo root.

**Two files carry direct commercial weight for the Define body-measurement pipeline:** `relevance_measurement/42_commercial_licence.md` and `relevance_measurement/43_accuracy_vs_tolerance.md`. Everything else builds the understanding needed to trust those two answers.
